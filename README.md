# 项目名称
stm32两轮平衡小车

## 项目简介
本项目基于STM32F103C8T6核心板，通过MPU6050传感器获取陀螺仪数据，使用PID算法控制直流电机，实现自主平衡，并可通过蓝牙模块实现远程遥控。

## 硬件清单
- STM32F103C8T6核心板
- MPU6050传感器模块
- TB6612FNG电机驱动模块
- 直流减速电机与车轮
- 锂电池供电模块
- JDY-31蓝牙模块
- OLED屏幕
- 降压芯片

## 软件依赖
- 开发环境：STM32CubeIDE
- 主要库：HAL库

## 使用说明
1. 自行连接电路，组装小车。
2. 使用IDE编译并烧录代码到STM32。
3. 开启电源给小车供电，测试其平衡状态。
